📜 SK eSports Official Rules

🚫 1. Anti-Cheat & Fair Play Policy

Hacking, config modification, auto-aim, ya kisi bhi tarah ke third-party tools ka use strict banned hai. Agar koi player cheat karte pakda gaya, toh uska account immediate **Permanent Block** kar diya jayega aur koi refund nahi milega.

📱 2. Allowed Devices

Sirf normal Mobile/Android/iOS devices allowed hain. **PC Emulators, Tablets, aur iPads strictly NOT allowed** hain. Agar aapne iPad se match join kiya toh point count nahi hoga.

🔑 3. Custom Room Timings

Match start hone se thik **15 minute pehle** aapke dashboard par Room ID aur Password live ho jayega. Sahi time par slot me baithna player ki zimmedari hai, late hone par game start ho jayega aur coin back nahi honge.

🎖️ 4. Wrong IGN / UID Entry

Registration form me aapka **In-Game Name (IGN)** aur **UID** ekdum sahi hona chahiye. Agar custom room me kisi doosre naam ka player aaya, toh use kick kar diya jayega.

💰 5. Payouts & Prize Distribution

Match khatam hone ke baad admin match results verify karega. Verify hone ke baad win kiye gaye coins **2 ghante ke andar** bache ke passbook me automatic add ho jayenge.
